## 📅 Phase 1：基礎建立（Week 1–2）

### Week 1：環境與概念

**目標**：建立開發環境、理解核心概念、形成學習習慣

#### 技術目標

- [ ] **開發環境建置**
  - [ ] 安裝 Node.js (v18+)、npm
  - [ ] 安裝 Hardhat 或 Foundry
  - [ ] 配置 MetaMask，取得測試網 ETH（Sepolia faucet）
  - [ ] 設定 VS Code 插件（Solidity、Prettier）

- [ ] **區塊鏈基礎**
  - [ ] 理解區塊結構：區塊、Hash、Merkle Tree
  - [ ] 理解交易生命週期：簽名 → 廣播 → 確認
  - [ ] 理解 Gas 機制與手續費計算
  - [ ] 理解 EOA 與 Contract Account 的差異

- [ ] **AI 基礎**
  - [ ] 理解 LLM 運作原理：Token、Embedding、Attention
  - [ ] 掌握 Prompt Engineering 基本技巧
  - [ ] 取得 Claude API Key，發送第一個 API 請求
  - [ ] 理解 Temperature、Top-p 等參數的意義

## 💡 學習原則

1. **每天至少 Commit 一次** — 即使只有一行筆記，也要推送
2. **實作優先** — 看文件不如動手寫，寫完再回頭理解
3. **記錄失敗** — 出錯的過程比成功更值得記錄
4. **社群優先** — 遇到問題先問社群，一個人走得快，一群人走得遠
5. **保護私鑰** — 永遠不在公開 Repo 或聊天室分享私鑰
